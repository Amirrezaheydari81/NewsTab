# نیوز تب

![cover](cover.jpg)

افزایش کروم صفحه تب جدید را با فیدهای RSS دلخواه شما به یک مرکز خبری شخصی‌سازی شده تبدیل می‌کند.

![Chrome Extension](https://img.shields.io/badge/Chrome-Extension-blue)
![Manifest V3](https://img.shields.io/badge/Manifest-V3-green)
![License MIT](https://img.shields.io/badge/License-MIT-lightgrey)
![Version](https://img.shields.io/badge/Version-1.1.0.0-purple)

## ویژگی‌ها

- **فیدهای RSS نامحدود** — هر تعداد فید دلخواه اضافه و مدیریت کنید
- **ترجمه خودکار فارسی** — تیترها را به فارسی ترجمه می‌کند
- **حالت کم مصرف** — برای سیستم‌های ضعیف‌تر با کاهش همزمانی، تعداد آیتم و فاصله به‌روزرسانی
- **مرتب‌سازی بر اساس زمان** — جدیدترین‌ها را اول ببینید
- **جستجو در تیترها** — فیلتر سریع در بین همه فیدها
- **کاهش فشار سیستم** — با عدم فچ کردن خودکار در باز شدن تب و استفاده از کش حافظه
- **پشتیبانی از گروه‌بندی** — فیدهای مرتب را در گروه‌های جداگانه Organized ببینید
- **نقل قول انگیزشی** — Quote روزانه از API برای انگیزه بیشتر
- **قیمت‌های لحظه‌ای** — نمایش قیمت طلا و دلار با ویجت tgju
- **منوی آیکونی** — دسترسی سریع به همه تنظیمات با طراحی شیشه‌ای
- **دکمه اضطراری پاکسازی کش** — پاکسازی کامل کش با یک کلیک
- **Tooltip سفارشی** — راهنمای hover برای همه دکمه‌ها
- **Auto-retry** — تلاش مجدد خودکار برای فیدهای timeout
- **رابط کاربری مینیمال** — طراحی شیشه‌ای و تم تیره v2

## نصب

### روش ۱: نصب مستقیم (Developer Mode)

1. فایل‌های پروژه را کلون یا دانلود کنید
2. در کروم به آدرس `chrome://extensions/` بروید
3. گزینه **Developer mode** را فعال کنید
4. روی **Load unpacked** کلیک کنید و پوشه‌ی `dist/` (یا پوشه اصلی برای توسعه) را انتخاب کنید

### روش ۲: بیلد برای انتشار

```bash
npm install
npm run build
```

فایل‌های تولید شده در پوشه‌ی `dist/` آماده‌ی بارگذاری در مرورگر هستند.

## توسعه

### پیش‌نیازها

- Node.js >= 18
- npm

### اسکریپت‌ها

| دستور | توضیحات |
|--------|---------|
| `npm run build` | بیلد production با فایل minified |
| `npm run watch` | بیلد و تماشای تغییرات (در حال توسعه) |
| `npm test` | اجرای تست‌های واحد |

### تست‌ها

پروژه شامل تست‌های واحد برای منطق اصلی است:

```bash
npm test
```

تست‌ها با استفاده از `jsdom` و `assert` نوشته شده‌اند و شامل موارد زیر هستند:
- `timeAgo` — فرمت‌سازی زمان
- `escapeHtml` - جلوگیری از XSS
- `sortItems` / `sortFeeds` / `feedLatestDate` — مرتب‌سازی
- `groupKeyOf` — گروه‌بندی فیدها
- `itemsHtml` — تولید HTML آیتم‌ها
- `errorMessage` — پیام‌های خطا
- `applyLowPowerMode` — حالت کم مصرف
- `toggleSort` / `toggleQuotes` / `toggleTickers` / `toggleMenu` — toggle ها
- `persist*` — توابع ذخیره‌سازی

### افزودن فید جدید

از طریق پنل تنظیمات (آیکون چرخ‌دنده) می‌توانید فیدهای جدید اضافه، حذف یا ترتیب دهید.

## تغییرات اخیر 1.1.0.0

- **رابط کاربری شیشه‌ای کامل**: اعمال Glassmorphism به تمام کامپوننت‌های اصلی شامل actions، منو، پنل تنظیمات و کارت‌ها
- **منوی آیکونی کامل**: تبدیل منوی همبرگری به کامپوننت عمودی آیکونی با طراحی دایره‌ای
- **دکمه اضطراری پاکسازی کش**: دکمه قرمز برای پاکسازی سریع کل کش
- **Tooltip سفارشی**: سیستم tooltip شیشه‌ای برای تمام دکمه‌های منو با قرارگیری هوشمند
- **رفع مشکل layering**: رفع مشکل نمایش منو پشت لیست RSS با مدیریت صحیح z-index
- **بهبود استایل دکمه‌ها**: اضافه شدن کلاس `.icon-btn` با طراحی شیشه‌ای یکسان برای همه دکمه‌ها
- **Badge نسخه**: نمایش نشانگر v2 در برند محصول
- رفع باگ رفرش: بهبود اطمینان به‌روزرسانی همه فیدها با بازگرداندن Promise از `enqueueFetch`
- Auto-retry برای خطای timeout: تلاش مجدد خودکار بعد از ۵ ثانیه
- اضافه شدن قیمت‌های لحظه‌ای طلا و دلار با ویجت tgju
- اضافه شدن نقل قول انگیزشی با ترجمه خودکار فارسی
- اضافه شدن منوی همبرگری برای دسترسی سریع
- اضافه شدن toggle برای نمایش/مخفی کردن قیمت‌ها و نقل قول
- اضافه شدن حالت کم مصرف برای سیستم‌های ضعیف‌تر
- مرتب‌سازی کارت‌ها بر اساس جدیدترین به‌روزرسانی
- عدم بارگذاری خودکار فیدها در باز شدن تب (کاهش فشار CPU)
- دکمه‌های action با tooltip و وضعیت فعال/غیرفعال
- لوگوی شخصی‌سازی در بخش برند
- اضافه شدن تست‌های واحد (33 تست پاس شده)
