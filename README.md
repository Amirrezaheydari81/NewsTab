# نیوز تب

افزایش کروم صفحه تب جدید را با فیدهای RSS دلخواه شما به یک مرکز خبری شخصی‌سازی شده تبدیل می‌کند.

![Chrome Extension](https://img.shields.io/badge/Chrome-Extension-blue)
![Manifest V3](https://img.shields.io/badge/Manifest-V3-green)
![License MIT](https://img.shields.io/badge/License-MIT-lightgrey)

## ویژگی‌ها

- **فیدهای RSS نامحدود** — هر تعداد فید دلخواه اضافه و مدیریت کنید
- **ترجمه خودکار فارسی** — تیترها را به فارسی ترجمه می‌کند
- **حالت کم مصرف** — برای سیستم‌های ضعیف‌تر با کاهش همزمانی، تعداد آیتم و فاصله به‌روزرسانی
- **مرتب‌سازی بر اساس زمان** — جدیدترین‌ها را اول ببینید
- **جستجو در تیترها** — فیلتر سریع در بین همه فیدها
- **کاهش فشار سیستم** — با عدم فچ کردن خودکار در باز شدن تب و استفاده از کش حافظه
- **پشتیبانی از گروه‌بندی** — فیدهای مرتب را در گروه‌های جداگانه Organized ببینید
- **رابط کاربری مینیمال** — طراحی شیشه‌ای و تم تیره

## نصب

### روش ۱: نصب مستقیم (Developer Mode)

1. فایل‌های پروژه را کلون یا دانلود کنید
2. در کروم به آدرس `chrome://extensions/` بروید
3. گزینه **Developer mode** را فعال کنید
4. روی **Load unpacked** کلیک کنید و پوشه‌ی `dist/` (یا پوشه اصلی برای توسعه) را انتخاب کنید

### روش ۲: بیلد برای انتشار

```bash
npm install
npm run build
```

فایل‌های تولید شده در پوشه‌ی `dist/` آماده‌ی بارگذاری در مرورگر هستند.

## توسعه

### پیش‌نیازها

- Node.js >= 18
- npm

### اسکریپت‌ها

| دستور | توضیحات |
|--------|---------|
| `npm run build` | بیلد production با فایل minified |
| `npm run watch` | بیلد و تماشای تغییرات (در حال توسعه) |

### افزودن فید جدید

از طریق پنل تنظیمات (آیکون چرخ‌دنده) می‌توانید فیدهای جدید اضافه، حذف یا ترتیب دهید.

## تغییرات اخیر

- اضافه شدن حالت کم مصرف برای سیستم‌های ضعیف‌تر
- مرتب‌سازی کارت‌ها بر اساس جدیدترین به‌روزرسانی
- عدم بارگذاری خودکار فیدها در باز شدن تب (کاهش فشار CPU)
- دکمه‌های action با tooltip
- لوگوی شخصی‌سازی در بخش برند
